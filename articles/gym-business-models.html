健身房商業模式 / 產業分析

健身房商業模式全解析：俱樂部、工作室、CrossFit 到計次制

Simon Chi 季祥・四季健身創辦人 2026年4月 6 分鐘閱讀

我在健身產業超過十年，從教練、到創辦自己的 CrossFit 場館、到輔導其他健身業者，我看過很多模式成功，也看過很多模式死掉。這篇文章不是教科書式的分析，而是我真正在這個行業裡打滾過的觀察。

台灣健身市場目前的四個主要商業模式：大型俱樂部、個人工作室（一對一 PT 為主）、CrossFit / 功能性訓練 Box、計次制大型健身房（俗稱「一分鐘一塊錢」模式）。它們的收入邏輯、成本結構、和生存關鍵截然不同。

模式一：大型健身俱樂部

代表：健身工廠、World Gym、加州健身（已停業）等。

收入邏輯：以月費或年費為主，靠大量會員基數攤平固定成本。部分收入來自 PT 課程的抽成。核心假設是：大部分辦了卡的人不會天天來，場地使用率遠低於 100%，所以場館可以賣出遠超過容量上限的會員數。

成本結構：高固定成本（大型場地租金 + 設備投資 + 人事費）。規模是護城河，單店難以獲利，連鎖才有議價能力和品牌效益。

生存關鍵：持續的會員招募能力（行銷和業務）、能源管理（大型健身房電費是大頭）、PT 課程收入的佔比優化。

現實問題：這個模式的天敵是低使用率正當化了低訂價的競爭。當一張健身卡只要 1,200 元一個月，消費者對品質的期望也降低了，你就很難在這個市場做出高端定位。加上設備折舊和場地成本，大型俱樂部的利潤率通常並不高，靠的是量。

模式二：個人工作室（一對一 PT）

代表：大量的獨立 PT 工作室，通常 1-3 個教練，面積 30-100 坪。

收入邏輯：每小時高單價的一對一課程。收費通常在 NT$1,500-3,500 / 小時。收入完全依賴教練的時間，是典型的「用時間換錢」模式。

成本結構：低固定成本（小場地、少設備）。創業門檻低，但收入有天花板：一個教練一天能帶的課數有限（通常 6-8 堂是上限）。

生存關鍵：客源穩定度和客戶黏著性。一對一 PT 的流失率高，因為學員達到目標後容易停課，或因為生活變化而中斷。有持續轉介的社群和口碑，才能維持。

現實問題：工作室模式很難規模化，因為你的產品就是教練本人的時間，無法複製。很多優秀的 PT 都在這個瓶頸上卡關：業績好，但想擴大就要多招教練，但管理多個教練的複雜度讓多數教練創業者吃不消。

模式三：CrossFit Box / 功能性訓練場館

代表：台灣現有的數十家 CrossFit Affiliate，包括我的四季健身 4SC。

收入邏輯：以月費（Membership）為基礎，提供團體課程。收費通常在 NT$2,500-5,000 / 月（無限課次）。部分 Box 也提供個人訓練和特殊課程。

成本結構：中等固定成本（場地租金 + 設備投資 + 教練薪資）。CrossFit 的空間設計比大型健身房有效率，同樣的面積可以服務更多課次。CrossFit 官方每年收取 Affiliate 費用（約 USD 3,000）。

生存關鍵：社群黏著度。CrossFit 最強的護城河不是器材，而是一起訓練的人。學員留在一個 Box，通常因為「這裡的人」，而不只是課程本身。社群建立能力是 CrossFit 創業者最核心的競爭力。

現實問題：月費模式的會員上限受到空間和課表容量限制。要增加收入，要麼增加課次（增加成本）、要麼提高單價（需要品牌支撐）、要麼增加多元收入（賽事、工作坊、PT）。我在 4SC 同時做了賽事和教育，是增加收入天花板的方式之一。

模式四：計次制大型健身房（一分鐘一塊錢模式）

這個模式的代表是部分中型健身房或 24 小時健身房，以「時間計費」或「單次計費」為主，不需要辦卡或簽合約。

收入邏輯：降低進入門檻（不用預付大筆費用），吸引偶爾來的使用者和不想綁定的消費者。單次收費通常在 NT$80-200 之間，部分採時間計費。

成本結構：規模化才能生存，需要相對完整的設備和 24 小時或長時段開放，人事成本控制嚴格（部分採無人值守或極少服務人員）。

生存關鍵：地點、交通便利性、使用率。這個模式靠的是流量，需要大量的來客數才能支撐固定成本。

現實問題：客戶黏著性極低，沒有社群、沒有服務、沒有人際關係。訓練品質完全靠學員自律，流失率高。這個模式對健身文化的培養沒有貢獻，是競爭最激烈、最難做出差異化的模式。

四種模式的比較表

用幾個維度快速比較：

  • 創業門檻：工作室 < Box < 計次制 < 大型俱樂部
  • 收入天花板：工作室 < Box < 計次制 < 大型俱樂部
  • 客戶黏著性：計次制 < 大型俱樂部 < 工作室 < Box
  • 對教練能力的依賴：Box < 大型俱樂部 < 計次制（無依賴） < 工作室（完全依賴）
  • 差異化空間：計次制最小，工作室和 Box 最大

我的觀察：什麼模式在台灣現在最有機會？

我不認為有一個「最好」的模式，但有幾個我觀察到的趨勢：

「社群型」訓練場館的空間在擴大。後疫情的消費者更在意人際連結和歸屬感，CrossFit Box 和社群型工作室這種模式，在「健身同時交到真正的朋友」這個需求上，有獨特的競爭優勢。

純一對一 PT 的壓力在增加。愈來愈多科技工具（AI 訓練 App、線上課程）在替代部分的 PT 功能，純粹依靠體能指導的 PT 工作室，如果不往更高端或更專業的方向走，會越來越難做出差異。

大型俱樂部的中間地帶很危險。不夠大（沒有規模成本優勢），也不夠小（沒有個性化服務），中型俱樂部是這個市場裡最難做的尺寸。

✦ 給想進場的人

如果你是準備創業的教練，最重要的問題不是「我要開哪種健身房」，而是「我能長期留住的客戶，是哪一類人？他們需要的是什麼？」商業模式是你的答案，不是起點。先想清楚你要服務誰，模式自然就跟著出來了。
